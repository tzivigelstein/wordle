Wordle (ES)

      Cómo jugar

      Adivina la plabra oculta en seis intentos.

      Cada intento debe ser una palabra válida de 5 letras.

      Despues de cada intento el color de las letras cambia para mostrar qué tan cerca estás de acertar la palabra.

      Ejemplos

      G
      A
      T
      O
      S

      La letra G está en la palabra y en la posición correcta.

      V
      O
      C
      A
      L

      La letra C está en la palabra pero no en la posición correcta.

      C
      A
      N
      T
      O

      La letra O no está en la palabra.

      Puede haber letras repetidas. Las pistas son independientes para cada letra.

      ¡Una palabra nueva cada día!
